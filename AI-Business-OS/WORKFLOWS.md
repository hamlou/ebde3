## WORKFLOW 1: BUSINESS DISCOVERY ENGINE
- **Trigger:** Daily
- **Process:** Market Intel scans → Creates Opportunity Report → Business Analyst evaluates → Devil's Advocate attacks → Executive Reviewer decides
- **Output:** Opportunity database updated.

## WORKFLOW 2: NEW BUSINESS CREATION
- **Trigger:** Approved opportunity
- **Phase 1 (Validation):** Test demand before building.
- **Phase 2 (MVP Build):** Product Strategist + Software Eng + Automation Eng build product.
- **Phase 3 (Launch):** Acquire first users.
- **Phase 4 (Optimization):** Measure and improve continuously.

## WORKFLOW 3: CUSTOMER ACQUISITION ENGINE
- **Process:** Research audience → Choose channels → Create content/offers → Launch experiments → Measure results → Optimize

## WORKFLOW 4: CONTENT MACHINE
- **Process:** Research → Generate → Review (QA) → Publish → Analyze

## WORKFLOW 5: AUTOMATION AUDIT
- **Trigger:** Weekly
- **Process:** Identify repetitive work → Estimate automation value → Build automation → Measure saved time

## WORKFLOW 6: COMPETITOR INTELLIGENCE
- **Trigger:** Weekly
- **Process:** Monitor products/pricing/features → Analyze works/fails → Update memory

## WORKFLOW 7: FAILURE RECOVERY
- **Trigger:** Any major failure
- **Process:** Detect → Contain → Analyze root cause → Create solution → Test → Document lesson → Update memory

## WORKFLOW 8: WEEKLY CEO MEETING
- **Participants:** CEO, Finance, Growth, Product, Engineering, Devil's Advocate
- **Agenda:** Financials, Customers, Product, Tech, Risks, Priorities
- **Output:** Weekly Strategy Report

## WORKFLOW 9: MAKE.COM AUTOMATION PRINCIPLE
- **Process:** Trigger → Data Collection → AI Processing → Decision → Action → Logging → Review
- **Requirements:** Error handling, Monitoring, Documentation, Recovery plan

## WORKFLOW 10: MCP TOOL USAGE
- **Before adding tool ask:** Does it increase capability? Save time? Reduce cost? If no → Do not add.

## FINAL WORKFLOW RULE
The company should continuously move through:
Discover → Validate → Build → Sell → Measure → Improve
Forever.
